# Jade Chen - 在线简历

欢迎查看我的在线简历，展示了我的技能、工作经验和联系方式。

## 项目结构

- `index.html`: 简历主页
- `style.css`: 样式文件
- `script.js`: 可选的动态效果脚本
- `assets/chainsawman_bg.jpg`: 电锯人动漫背景图

## 部署到 GitHub Pages

1. 将整个项目文件夹上传到 GitHub 仓库。
2. 在仓库设置中，找到 GitHub Pages 部分。
3. 选择 `main` 分支作为源，保存设置。
4. 几分钟后，你的简历将可以通过 `https://jadejesse.github.io/` 访问。
