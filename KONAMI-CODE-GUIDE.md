# 🎮 Konami Code 使用指南

## 什么是 Konami Code？
Konami Code 是一个经典的游戏彩蛋，最早出现在1986年的游戏中。现在它成为了你简历的一个酷炫隐藏功能！

## 如何使用？

### 步骤 1: 打开简历页面
打开 `index.html` 文件在浏览器中

### 步骤 2: 确保页面激活
点击页面任意位置，确保页面是激活状态（不是在其他标签页）

### 步骤 3: 按键顺序
按照以下顺序按键盘上的键：

```
1. ↑ (向上箭头键)
2. ↑ (向上箭头键) - 再按一次
3. ↓ (向下箭头键)
4. ↓ (向下箭头键) - 再按一次
5. ← (向左箭头键)
6. → (向右箭头键)
7. ← (向左箭头键) - 再按一次
8. → (向右箭头键) - 再按一次
9. b (字母b键 - 小写就可以)
10. a (字母a键 - 小写就可以)
```

### 视觉提示
- 每按对一个键，屏幕中央会显示进度：`🎮 3/10 - ↑ ↑ ↓`
- 如果按错了，会自动重置，从头再来就好
- 浏览器控制台（F12）也会显示进度

### 成功后会发生什么？
✨ **彩虹模式激活！**
- 整个页面会开始变换颜色（彩虹效果）
- 会播放一个庆祝音效
- 卡片会有轻微的动画效果
- 刷新页面可以关闭效果

## 💡 小技巧

### 如果不起作用：
1. 确保页面已经完全加载
2. 点击页面任意位置确保焦点在页面上
3. 按键不要太快，正常速度就好
4. 确保使用的是键盘上的方向键（不是数字键盘）
5. 字母 b 和 a 不区分大小写

### 查看进度：
- 按 F12 打开开发者工具
- 切换到 Console（控制台）标签
- 每按对一个键都会显示进度
- 按错了也会有提示

## 🎯 快速参考
