# 🚀 快速开始指南

## 立即测试所有功能！

### 1️⃣ 打开简历
双击 `index.html` 文件，在浏览器中打开

### 2️⃣ 测试 Konami Code（最酷的功能！）

**操作步骤：**
1. 确保简历页面在浏览器中打开
2. 点击页面任意位置（确保页面是激活的）
3. 按键盘上的这些键：

```
键盘操作：
第1步: 按 ↑ (向上箭头)
第2步: 再按 ↑ (向上箭头)
第3步: 按 ↓ (向下箭头)
第4步: 再按 ↓ (向下箭头)
第5步: 按 ← (向左箭头)
第6步: 按 → (向右箭头)
第7步: 再按 ← (向左箭头)
第8步: 再按 → (向右箭头)
第9步: 按字母键 b
第10步: 按字母键 a
```

**记忆口诀：** 上上下下左右左右BA

**你会看到：**
- 屏幕中央显示进度（例如：🎮 5/10 - ↑ ↑ ↓ ↓ ←）
- 按对了会有提示
- 按错了会重置，从头再来
- 全部按对后 → 🌈 彩虹模式激活！

## 🎯 常见问题

### Q: Konami Code 不起作用？
**A:** 
- ✅ 确保页面已完全加载
- ✅ 点击页面确保焦点在页面上
- ✅ 使用键盘上的方向键（不是数字键盘的方向键）
- ✅ 字母 b 和 a 用小写就可以
- ✅ 按键不要太快，正常速度

### Q: 怎么知道我按对了？
**A:** 
- 屏幕中央会显示进度
- 浏览器控制台（F12）会显示 ✅ 提示
- 按错了会显示 ❌ 并重置

### Q: 音乐太大声？
**A:** 
- 音量已设置为15%（很轻）
- 可以在 `enhanced-features.js` 中修改 `volume: 0.15`

### Q: 如何关闭彩虹模式？
**A:** 
- 刷新页面（F5）即可
