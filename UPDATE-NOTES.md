## 🔧 自定义选项

### 修改音乐：
```javascript
// Resume页面 (enhanced-features.js)
this.audio.src = 'YOUR_LOFI_URL';

// Home页面 (home-script.js)
this.audio.src = 'YOUR_CITYPOP_URL';
```

### 修改音量：
```javascript
volume: 0.15, // 改为 0.0-1.0 之间的值
```

### 修改Party照片：
```javascript
overlay.innerHTML = '<img src="YOUR_IMAGE.png" alt="Special" class="party-image">';
```

### 修改提示框颜色：
在 `enhanced-features.css` 中修改：
```css
.easter-egg-hints {
  background: linear-gradient(135deg, YOUR_COLOR_1, YOUR_COLOR_2);
}
```
